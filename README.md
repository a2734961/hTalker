### 开发流程

- 项目搭建与基础模块准备
- 数据库模型设计
- 聊天基础模块搭建（登录注册、关注、联系人）
- 数据库操作与数据仓库封装
- 实现消息通讯（聊天）
- 群聊天的实现（多成员通知送达）
- 聊天扩展（表情、语音、图片）

### IM四大协议

- IMPP：即时信息和空间协议
- XMPP： 可扩展通讯和表示协议
- SIMPLE(SIP)：针对即时信息和空间平衡的扩充协议
- PRIM：空间和即时信息协议（已未使用）
